Dashboards

We have created a set of dashboards containing some essential, pre-defined metrics, to help you get started on your Overwatch journey. These are meant to be a learning resource for you to understand the data model, as well as a practical resource to help you get value out of Overwatch right away. As a first iteration, these are notebook-based dashboards, in the future we’ll have these available in DBSQL as well.

Available Dashboards

Start here, this is your initial overall view of the state of the workspaces you monitor with Overwatch
